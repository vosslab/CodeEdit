# Third-party notices

## KDE KSyntaxHighlighting XML subset

This repository vendors the KDE KSyntaxHighlighting XML syntax-definition
collection from a pinned upstream snapshot for the plain-editor syntax import
path.

Files currently in use:

- all `Packages/CodeEditSyntaxDefinitions/Sources/CodeEditSyntaxDefinitions/Resources/Vendor/Kate/*.xml`

The files preserve their embedded upstream copyright and license metadata.
The repository keeps a pinned upstream snapshot rather than a live fetch. The
vendored files are not all under the same license, so each file's own header
remains the source of truth.

See `ThirdParty/KSyntaxHighlighting/UPSTREAM.md` for the import policy and
`ThirdParty/KSyntaxHighlighting/LICENSES/MIT.txt` for the MIT text used by one
of the vendored files.
